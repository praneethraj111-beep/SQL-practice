# Agricultural Management System (SQL)

## Project Overview
This project is a comprehensive database system designed to manage and analyze agricultural operations. It tracks crop growth cycles, expenditures, and yields to provide data-driven insights into farm productivity and financial performance.

## Tech Stack
* **Database**: PostgreSQL
* **Interface**: DBeaver
* **Language**: SQL (DDL & DML)

## Project Structure
* `sql/`: Contains table schema definitions and complex analysis queries.
* `data/`: CSV files used for data seeding and analysis.
* `README.md`: Project documentation and insights.

## Key Insights & Queries
The system provides answers to critical business questions, such as:
1. **Total Investment per Crop**: Aggregating expenditures across different growth cycles.
2. **Yield Performance**: Calculating average yields to identify the most productive crops.
3. **Budget Tracking**: Monitoring costs against specific crop categories.

## How to Use
1. Run the scripts in the `sql` folder to create your tables.
2. Import the datasets from the `data` folder.
3. Execute analysis queries to generate reports.
