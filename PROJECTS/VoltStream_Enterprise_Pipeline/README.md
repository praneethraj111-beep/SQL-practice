\# VoltStream: EV Fleet \& Energy Analytics Dashboard



\## 📌 Project Overview

A professional Power BI dashboard designed to provide actionable intelligence for EV fleet management. This project transforms raw trip data into a decision-support tool to optimize energy consumption, monitor operational costs, and evaluate driver performance.



\## 🚀 Key Features \& Problem Solving

\* \*\*Segment Intelligence:\*\* Interactive filtering across 2W, Taxi, SUV, and Luxury segments to compare performance.

\* \*\*Efficiency Metrics:\*\* Real-time tracking of Average Wh/km to identify the most energy-efficient vehicle models.

\* \*\*Operational Cost Analysis:\*\* Localized financial reporting (₹/km) to monitor budget allocation and spend trends.

\* \*\*Driver Behavior:\*\* Visual leaderboard to identify high-efficiency drivers and coaching opportunities.

\* \*\*Trend Monitoring:\*\* Time-series analysis of running costs to detect seasonal or technical anomalies.



\## 🛠️ Technical Stack

\* \*\*Data Tool:\*\* Power BI Desktop

\* \*\*Data Modeling:\*\* Star Schema / Flat File (EV Fleet Logs)

\* \*\*Localization:\*\* Indian Rupee (₹) currency formatting and metric renaming for executive clarity.
