\# Real Estate Data Operations \& Analytics



An end-to-end Data Engineering project simulating high-value real estate transactions for global investors.



\## 🏗️ Technical Stack

\- \*\*Database:\*\* PostgreSQL (Schema Design: Star Schema)

\- \*\*Middleware:\*\* SQL Views, CTEs, and Window Functions

\- \*\*Visualization:\*\* Power BI Desktop

\- \*\*Version Control:\*\* Git



\## 🛠️ Project Architecture

1\. \*\*DDL/DML:\*\* Designed a relational database with constraints and automated triggers for property status updates.

2\. \*\*Business Logic:\*\* Authored SQL Views to calculate Net Equity and Brokerage Yields.

3\. \*\*BI Layer:\*\* Connected Power BI to PostgreSQL via Import mode to visualize investor performance.



\## 📊 Key Insights

\- \*\*Portfolio Value:\*\* $819M+ across major global hubs (Madrid, London, Lisbon).

\- \*\*Efficiency:\*\* Monitored brokerage yields to stay under a 5% "leakage" threshold.

\- \*\*Diversification:\*\* Real-time analysis of Villa vs. Penthouse investment split.



\## 🚀 How to Run

1\. Execute the scripts in `/sql/` to build the database.

2\. Connect `Real\_Estate\_Final.pbix` to your local PostgreSQL instance.
